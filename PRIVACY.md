# Privacy Policy (Simplified)


This tool acts as a Dify node or agent. It only forwards your input text or images to the ModelScope API-Inference for text‑to‑image or image‑to‑image generation, and returns the generated image to you. **We do not store your text descriptions, uploaded images, or generated images on any server.**

## 1. How We Handle Your Data
- **Collection:** When you invoke this tool via Dify, we receive the text description or image you provide.
- **Use:** The data is forwarded solely to the ModelScope API (https://api-inference.modelscope.cn) to perform image generation, and the result is returned to you.
- **Storage:** This tool does not use a database. No input or output data is stored. All data exists only temporarily during request processing and is automatically cleared afterwards.

## 2. Data Sharing
To provide the core functionality, your text and/or image are transmitted to **ModelScope** (a third‑party API service). ModelScope will process the data according to its own privacy policy. We recommend that you review ModelScope’s privacy policy.

## 3. Data Security
We encrypt your data using HTTPS during transmission to help protect it from unauthorized access or tampering.

## 4. Your Rights
Since we do not store any data, there is no need to request deletion. You may save the generated results on your own and are responsible for your use of the tool.
